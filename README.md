# Bhavishya Kaushik — Portfolio Website

A modern, dark-themed developer portfolio built with pure HTML, CSS, and JavaScript.

## 🚀 Live Demo

> Hosted via GitHub Pages: **https://bhavishya208.github.io/Bhavishya-Portfolio-/**

## ✨ Features

- 🎨 Dark terminal-inspired design with neon accents
- 🖱️ Custom animated cursor with ring follow effect
- 💻 Animated terminal card in hero section
- 🃏 Project cards with glow effects & performance stats
- 🎞️ Scroll-reveal animations on all sections
- 📱 Fully responsive (mobile + desktop)
- ⚡ Zero dependencies — pure HTML/CSS/JS

## 📁 Project Structure

```
portfolio/
├── index.html       # Main portfolio page (single file)
└── README.md        # This file
```

## 🛠️ Tech Stack

- HTML5
- CSS3 (custom properties, grid, animations)
- Vanilla JavaScript (IntersectionObserver, RAF cursor)
- Google Fonts (Syne, DM Mono, Instrument Serif)

## 🌐 Deploy on GitHub Pages

1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Under **Source**, select `main` branch and `/ (root)` folder
4. Click **Save**
5. Your site will be live at `https://<your-username>.github.io/<repo-name>`
